# Сегментация клиентов телекоммуникационной компании

Разведочный анализ данных (EDA) и сегментация клиентов по возрастным группам для разработки персонализированных предложений.

## - Цель проекта
Выделить типичные группы клиентов на основе их поведения (звонки, SMS, расходы) и предложить рекомендации по работе с каждой группой.

## - Описание данных
Каждый клиент описан следующими признаками:
- Возраст  
- Среднемесячный расход  
- Средняя продолжительность разговоров  
- Количество звонков (днём, вечером, ночью, в другие города/страны)  
- Доля звонков на стационарные телефоны  
- Количество SMS за месяц  
- Дата подключения тарифа  

> Данные учебные — использованы в рамках программы профессиональной переподготовки «Аналитик данных» (ТГУ, 2025).

## - Используемые технологии
- Python  
- pandas  
- Matplotlib / Seaborn  
- Jupyter Notebook  
- Разведочный анализ данных (EDA)

## - Основные этапы
1. Загрузка и предобработка данных (проверка типов, дубликатов, аномалий)  
2. Создание новых признаков:
   - `Возрастная категория`: студент (19–24), аспирант (25–33), бизнесмен (34–56), знаток (57–70)
   - `Год`, `Месяц`, `День подключения`
3. Анализ поведения по возрастным группам: расходы, активность, предпочтения по времени суток  
4. Визуализация трендов и зависимостей  
5. Формулировка бизнес-выводов и рекомендаций

## - Ключевые выводы
- **Бизнесмены** — самая прибыльная группа: максимальный среднемесячный расход.  
- **Студенты** чаще используют SMS и звонят ночью.  
- Пик подключений пришёлся на **март 2023 года**.  
- Наблюдается положительная связь между длительностью разговоров и расходами.
